apiVersion: v1
kind: PersistentVolume
metadata:
  name: mongodb-pv
spec:
  capacity:
    storage: 1Gi  # Adjust as necessary
  accessModes:
    - ReadWriteMany  # NFS supports this
  persistentVolumeReclaimPolicy: Retain   # Added this so your data isn't deleted if the PVC is deleted
  nfs:
    server: 172.31.11.218  # Your NFS server pvt IP
    path: /mnt/nfs_share  # Path on your NFS server



apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: mongodb-pvc
  namespace: prod # This is correct
spec:
  accessModes:
    - ReadWriteMany  # Must match the PV's access mode
  resources:
    requests:
      storage: 1Gi  # Must match or be less than the PV's size
---

NOTE: pv and pvc are 1 to 1 relationship
====================================================================================================
why we use this because anythig changes in PV or PVC first delete the existing pv and pvc and redeploy.

1. Force-Remove the PV Lock.
Now that  the PV might still be stuck. Run this to clear the PV:
ubuntu@ip-172-31-22-141:~$ kubectl get pvc -n prod
NAME         STATUS        VOLUME      CAPACITY   ACCESS MODES   STORAGECLASS   VOLUMEATTRIBUTESCLASS   AGE
mongodbpvc   Terminating   mongodbpv   1Gi        RWX                           <unset>                 24h

if show this error 
STATUS
Terminating
Run this cmd.
C: kubectl patch pv mongodbpv -p '{"metadata":{"finalizers":null}}'

---------------------------------------------------------------------------------------------------------------------------------

2.Force-Remove the PVC Lock.
This will make the PVC disappear immediately.
ubuntu@ip-172-31-22-141:~$ kubectl get pv -n prod
NAME        CAPACITY   ACCESS MODES   RECLAIM POLICY   STATUS        CLAIM                VOLUMEATTRIBUTESCLASS   REASON   AGE
mongo-pv    1Gi        RWX            Retain           Terminating   prod/mongo-pvc                   <unset>              6d1h
mongodbpv   1Gi        RWX            Retain           Terminating   prod/mongodbpvc                  <unset>               24h

if show this error 
STATUS
Terminating
Run this cmd.
C: kubectl patch pvc mongodbpvc -n prod -p '{"metadata":{"finalizers":null}}'
